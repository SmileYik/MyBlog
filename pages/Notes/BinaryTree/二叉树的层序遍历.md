给你一个二叉树, 按照 **层序遍历** 返回节点值.

## 层序遍历

从根节点到树枝, 从左至右, 即一层一层从左至右遍历二叉树.

### 题解1

很容易想到广度优先搜索.

代码如下: 

``` java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
  public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> list = new ArrayList<>();
    List<Integer> subList = new ArrayList<>();
    Queue<TreeNode> queue = new LinkedList<>();
    if (root != null) {
      queue.add(root);
    }
    int size = 1;
    int nextSize = 0;
    while (!queue.isEmpty()) {
      root = queue.poll();
      subList.add(root.val);
      
      // 从左至右添加有效的子树节点到队列中.
      // 并且改变下一层节点的数量.
      if (root.left != null) {
        queue.add(root.left);
        ++nextSize;
      }
      if (root.right != null) {
        queue.add(root.right);
        ++nextSize;
      }
      
      // 记录这一层还剩多少节点.
      // 如果这一层的节点归0, 
      // 则进入下一层.
      --size;
      if (size == 0) {
        size = nextSize;
        nextSize = 0;
        list.add(subList);
        subList = new ArrayList<>();
      }
    }
    return list;
  }
}
```
