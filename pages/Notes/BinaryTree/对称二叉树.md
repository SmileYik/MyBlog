给定一个二叉树, 判断他是否镜像对称的.

例如，二叉树 `[1,2,2,3,4,4,3]` 是对称的。

        1
       / \
      2   2
     / \ / \
    3  4 4  3
 

但是下面这个 [1,2,2,null,3,null,3] 则不是镜像对称的:

        1
       / \
      2   2
       \   \
       3    3

### 题解1

用迭代方法:

``` java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
  public boolean isSymmetric(TreeNode root) {
    if (root == null) {
      return true;
    }
    // 用数组是因为数组传参时传的是地址.
    // 记录着是否对称.
    boolean[] flag = {true};
    isSymmetric(root.left, root.right, flag);
    return flag[0];
  }
  
  public void isSymmetric(TreeNode left, TreeNode right, boolean[] flag) {
    // 如果到目前为止是对称的,
    // 则继续进行以下操作.
    if (flag[0]) {
      // 如果left与right同时为null
      // 或者他两的值同时相同
      // 则进行下一步递归, 
      // 以判断它们的子树是否镜像对称.
      if ((left == null && right == null) 
          || (left != null && right != null && left.val == right.val) ) {
        if (left != null) {
          // 注意镜像对称那就意味着
          // 左子树的左枝叶与右子树的右枝叶相等
          // 左子树的右枝叶与右子树的左枝叶相等.
          isSymmetric(left.left, right.right, flag);
          isSymmetric(left.right, right.left, flag);
        }
      } else {
        flag[0] = false;
      }
    }
  }
}
```

### 题解2

迭代方法.

这里使用的是栈来记录节点.

同理可以使用队列来记录节点.

代码如下:

``` java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
  public boolean isSymmetric(TreeNode root) {
    Stack<TreeNode> stack = new Stack<>();
    // 这里先压入左子树
    // 再压入右子树
    // 这就意味着以下获取时,
    // 应该先给右子树赋值
    // 再给左子树赋值
    // 并且再次压入栈时,
    // 也应该按开头压入栈的顺序压入.
    stack.add(root.left);
    stack.add(root.right);
    TreeNode left;
    TreeNode right;
    while (!stack.isEmpty()) {
      right = stack.pop();
      left = stack.pop();
      if (right == null && left == null) {
        // 如果左子树根节点与右子树根节点都为null
        // 则直接跳过本次循环.
        continue;
      } else if (right != null && left != null && right.val == left.val) {
        // 如果左子树根节点与右子树根节点的值相等,
        // 那么依次按照顺序一对一对加入栈中.
        // 注意镜像对称那就意味着
        // 左子树的左枝叶与右子树的右枝叶相等
        // 左子树的右枝叶与右子树的左枝叶相等.
        stack.add(right.right);
        stack.add(left.left);
        stack.add(right.left);
        stack.add(left.right);
      } else {
        // 左子树与右子树不同, 则直接返回false
        return false;
      }
    }
    return true;
  }
}
```
